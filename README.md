# Архив «Гостинной»

Статический SPA-архив для GitHub Pages: `index.html`, `style.css`, `script.js` и папка `assets/`.

## Структура

- `#home` — Работы Забора, папка `assets/zabor/` (134 изображений)
- `#works` — Работы сообщества, папка `assets/community/` (344 изображений)
- `#cringe` — скрытая страница, папка `assets/cringe/` (27 изображений)

## Секретная страница

На странице `#works` в подвале есть поле ввода. Ключ: `кринж`.

Это не настоящая защита, потому что проект полностью статический: любой человек с доступом к репозиторию может увидеть файлы и код. Это именно скрытая страница для рофла, а не приватный раздел.

## Публикация на GitHub Pages

1. Создай репозиторий.
2. Залей содержимое этой папки в корень репозитория.
3. В настройках репозитория открой **Pages**.
4. Source: **Deploy from a branch**.
5. Branch: `main`, folder: `/root`.
6. Сохрани настройки и дождись публикации.

## Обновление архива

Проще всего добавлять новые картинки в нужную папку `assets/...` и дописывать их в массив `IMAGE_DATA` в `script.js`.
Если изображений станет очень много, удобнее вынести список в отдельный JSON и генерировать его скриптом.
